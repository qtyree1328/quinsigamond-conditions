# Lake Quinsigamond Annual Report Digests (2022–2025)

Structured digests of the City of Worcester Lakes & Ponds Program annual Lake Quinsigamond Water Quality Reports.

These power the **Annual reports** page (`reports.html`) of the dashboard.

## Files

- `../../data/reports.json` — merged machine-readable digest (per-year data + cross-year trends + scorecard).
- `../../data/report-gallery.json` — curated figure gallery manifest.
- Per-year markdown digests:

  - [`2022.md`](2022.md) — overall **Good**
  - [`2023.md`](2023.md) — overall **Good/Fair**
  - [`2024.md`](2024.md) — overall **Good/Fair**
  - [`2025.md`](2025.md) — overall **Good**

## Method

Each report's text and every page were read (text + figures). Ratings and min/max/peak numbers are transcribed from the report text and tables (high confidence). Seasonal chart shapes were read from report figures and are approximate. PFAS/industrial contaminants are tested on a ~3-year cycle (last full panel 2022, no results of concern); 2023–2025 reports carry no new PFAS values.
